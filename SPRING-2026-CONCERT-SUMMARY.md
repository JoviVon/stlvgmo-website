### Use the Flyer Centerpiece

1. Open the flyer in your browser: `http://localhost:3000/assets/Spring-2026-Flyer-Centerpiece.html`
2. To print or save as PDF:
   - Press `Ctrl+P` (Windows) or `Cmd+P` (Mac)
   - Select "Save as PDF" or your printer
   - Ensure "Background graphics" is enabled in print settings
   - Set paper size to Letter (8.5" x 11")

### Add a QR Code (Optional)

To add a real QR code to the flyer:

1. Generate a QR code for the ticket link: `https://givebutter.com/mY39pq`
2. Save the QR code image to `public/assets/`
3. Edit the flyer HTML and replace the `[QR CODE]` placeholder section with:
   ```html
   <img
     src="your-qr-code.png"
     alt="Ticket QR Code"
     style="width: 100%; height: 100%;"
   />
   ```
