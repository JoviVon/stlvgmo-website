# Gallery Images

This folder contains images that will be displayed in the STLVGMO website gallery.

## How to Add Images

Simply add your image files to this folder. The website will automatically detect and display them!

### Supported Formats

- `.jpg` / `.jpeg`
- `.png`
- `.gif`
- `.webp`

### Where Images Appear

1. **Homepage Carousel**: All images will cycle automatically every 5 seconds
2. **Gallery Page** (`/gallery`): All images displayed in a responsive grid

### Tips

- **File Names**: Use descriptive names (e.g., `mario-concert-2024.jpg`)
- **File Size**: Optimize images for web (recommended: under 2MB per image)
- **Dimensions**: Images will be automatically scaled to fit, but 1920x1080 or similar is recommended
- **Organization**: Images are sorted alphabetically by filename

### Example

```
public/assets/gallery/
├── concert-march-2024.jpg
├── rehearsal-group-photo.png
├── performance-mario-theme.jpg
└── orchestra-setup.jpg
```

All these images will automatically appear on your website!
